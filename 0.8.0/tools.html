The python-toolbox ships with a set of command line tools, whose entry point always is the command tbx. The commands are structured in a tree manner, and help is provided along with the command(s) no matter the nesting.

$ tbx --help

$ tbx command --help

$ tbx command subcommand --help

$ tbx command subcommand subsubcommand --help
